**つまづいたポイント**:
- 無限ループから抜け出せなくなった
- 原因: 条件式の書き方が間違っていた
- 解決方法: `while True`と`break`を使う方法に変更

**コード例**:
```python
def ask_choice(prompt: str, choices: set) -> str:
    """選択肢から入力を受け付ける"""
    while True:
        answer = input(prompt)
        if answer in choices:
            return answer
        print(f"❌ {choices}から選んでください")
```

**感想・メモ**:
- ヘルパー関数を作ることでコードが読みやすくなった

## 💡 重要な気づき

### プログラミング全般
- [ ] エラーメッセージは必ず最後まで読む

- [ ] 変数名は分かりやすく（タイポを防ぐ）

- [ ] インデントは必ず揃える（エラーの原因になる）
