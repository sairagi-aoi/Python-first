## 🎯 学習のコツ

### 1. 順番に進める
上から順番に学習することで、段階的にスキルアップできます。

### 2. 実際に動かす
読むだけでなく、必ず実行して動作を確認しましょう。

```bash
python fukusyu/kata.py
```

### 3. コードを変えてみる
- 変数名を変える
- 値を変える
- 条件を変える

自分で変更を加えることで理解が深まります。

### 4. わからなくても進む
最初は理解できなくても、後で復習すると「あ、こういうことか！」となることがあります。

### 5. メモを取る
ファイル内にコメントで気づきを書き込むのもおすすめです。

```python
# 2025-11-25: ここでつまづいた。〇〇が理由だった
```

## 📝 練習方法の例

### 初見のとき
1. ファイルを開いて全体を読む
2. 実行して動作を確認
3. コメントを読んで理解を深める

### 復習のとき
1. コードを見ずに同じ内容を書いてみる
2. 書き終わったら元のファイルと比較
3. 違いや間違いを確認

### 応用のとき
1. 似たような機能を自分で実装
2. 練習ファイルの内容を拡張してみる
3. 新しいアイデアを試してみる
